ABI Break Fix with Version Script
==================================

ABI Compatibility Demonstration:
Old client (linked to @VERS_1.0):
  V1: Processing single value: 10
  Result: 20

New client (linked to @@VERS_2.0):
  V2: Processing with multiplier: 10 * 3
  Result: 30

Default version:
  V2: Processing with multiplier: 10 * 3
  Result: 30

Version scripts maintain backward compatibility
Old binaries use @VERS_1, new binaries use @@VERS_2
